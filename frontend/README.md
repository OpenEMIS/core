# Frontend

This directory contains the unified frontend dashboard for the Ethiopian Educational SaaS platform. It is built using React.

## Getting Started

1.  Navigate to this directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm start
    ```
