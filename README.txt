OpenEMIS - Better Data. Better Education.

OpenEMIS <https://www.openemis.org> is an open source Education Management Information System (EMIS) application designed to manage education data.

You can download OpenEMIS <https://www.openemis.org/downloads> and install it on your own servers, or contact OpenEMIS <https://www.openemis.org/get-started> to arrange support and / or cloud hosting. 

OpenEMIS is widely used around the world in many countries to manage schools,  students and staff. 

OpenEMIS is provided freely as open source software, under the GNU General Public License <https://www.openemis.org/resources/agreements/license-agreement/>. 

OpenEMIS is written in PHP and JavaScript and uses an SQL database for storing  the data.    

See <https://support.openemis.org> to access the online knowledge bases.
